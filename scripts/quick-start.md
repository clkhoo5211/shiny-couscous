# Quick Start Guide

## Start Backend Server

```bash
cd backend
python3 -m uvicorn labuan_fsa.main:app --host 127.0.0.1 --port 8000 --reload
```

Open browser: http://localhost:8000/docs

## Start Frontend Server

```bash
cd frontend  
npm run dev
```

Open browser: http://localhost:3000
